# Donatepay WebSocket connector
## Setup
1. Create .env file

    ```python
    TOKEN = "YOUR-TOKEN-HERE"
    ```
    [Here you can get token](https://donatepay.ru/donation/notifications/)

    https://widget.donatepay.ru/alert-box/widget/[YOUR-TOKEN]

2. Install depencies
    ```sh
    pip install -r requirements.txt
   ```
3. Run it
    ```sh
    python main.py
   ```
